Media Queries to use

Properties

@ (min/max)-width: viewport width

@ (min/max)-height: viewport height

@ orientation: portrait(h>w) | landscape (w>h)

@ (min/max)-aspect-ratio: width/height

@ (min/max)-resolution: 72dpi | 100dpcm

Syntax/Punctuation

"Only" leaves out older browsers

media="only print and (color)"

"and" both parts must be true

media="only screen and (orientation: portrait)

"not" if untrue

media="not screen and (color)"

Comma separated selectors

- Any part can be true

media="print, scree and (min-width: 480px)"

Capability Detection

Don't Use

@media screen and (transform-3d){.transforms{}}

Resolution Units

Properties

> dpi: dots per inch (72, 96)

> dpcm: dots per centimeter (1dpcm = 2.54 dpi)

> dppx: dots per pixel (1dppx = 96 dpi) (default res of img)
